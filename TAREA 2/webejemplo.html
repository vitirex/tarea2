CSS: Sintaxis y Selectores

Sintaxis CSS

Un conjunto de reglas CSS consta de un selector y un bloque de declaracion:

  • El selector apunta hacia el elemento HTML que desea dar estilo

  • El bloque de declaracion contiene uno o mas declaraciones separadas por punto y coma
  • Cada declaracion incluye un nombre de propiedad CSS y un valor, separados por dos puntos.
  • Una declaración CSS siempre termina con un punto y coma, y los bloques de declaraciones están rodeados por llaves

    Los selectores CSS

    Los selectores CSS se utilizan para "encontrar" (o seleccionar) elementos HTML en función del nombre (del elemento), ID,clase, at
